pylibfreenect2.libfreenect2.setGlobalLogger¶

pylibfreenect2.libfreenect2.setGlobalLogger()¶

Same as void libfreenect2::getGlobalLogger(Logger*)

Parameters:

logger : Logger

Python side instance for libfreenect2::Logger*. If None, setGlobalLogger(NULL) will be called, i.e. logging is disabled. Default is None.
